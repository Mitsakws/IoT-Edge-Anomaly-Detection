# Edge AI Anomaly Detection in IoT Environments

## Overview
This repository contains the data analysis, Machine Learning experiments, and related documentation for my thesis project on **IoT Anomaly Detection**. The current focus is on developing a robust algorithm to detect unnatural thermal shocks (e.g., sudden window openings) while ignoring natural temperature gaps caused by room insulation.

## Current Stage: Algorithm Evaluation & Feature Engineering
To solve the "insulation false-positive" problem, we conducted an A/B/C/D experimental study comparing raw data against engineered dynamic features:
1. **Raw Data:** Models trained on absolute indoor/outdoor temperatures fail, as they penalize the building's thermal insulation.
2. **Feature Engineering:** We extract rate-of-change (`roc_10min`), rolling standard deviation, and `delta_thermal_gap`. 
3. **Robust Scaling:** We utilize `RobustScaler` (median/IQR) to ensure that true anomalies (spikes) are not suppressed.

## Repository Structure & Resources
- 📁 `/data`: Contains the `feeds.csv` dataset (resampled to 10-minute intervals).
- 📁 `/src`: Python scripts for data preprocessing, Feature Extraction, and the generation of comparison metrics.
- 📁 `/presentations`: Slide decks explaining the core problem, the methodology, and the experimental results.
- 📁 `/images`: High-resolution plots and visualizations generated by the models (Time-series, Scaler comparisons, Score distributions).
- 📁 `/papers`: Reference literature, including related IEEE papers on Edge Computing and ML resource scaling.
- 📁 `/demo`: Screenshots and brief video demonstrations showing the execution of the Python pipeline and terminal outputs.

## Next Steps (Hardware Deployment)
